# 🌍 Tripzen

Tripzen is a modern and responsive travel website designed to help users explore destinations, discover travel packages, and plan their journeys with ease.

## ✨ Features

* 🌎 Explore popular travel destinations
* 🏝️ Beautiful UI with modern design
* 📱 Fully responsive (mobile + desktop)
